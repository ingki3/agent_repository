# Agent Client

Agent Client는 1인 운영자가 다중 AI 에이전트(SimpleClaw, OpenClaw 등)와 모바일에서 텔레그램 호환 UX로 소통하며 작업을 제어할 수 있게 해주는 전용 클라이언트 앱입니다.

## 문서
- [Product Requirements Document (PRD)](./PRD.md)

## 기술 스택
- 프레임워크: React Native (Expo)
- 상태 관리: Zustand
- 로컬 저장소: SQLite
- 모바일 네이티브 연동: @react-native-voice/voice, expo-speech, SecureStore

*추후 빌드 및 실행 명령어가 업데이트될 예정입니다.*
